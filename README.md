# StudyShare

StudyShare là một ứng dụng chia sẻ tài liệu học tập, giúp sinh viên và người học dễ dàng trao đổi, quản lý và lưu trữ tài liệu. 

## Tính năng
- Quản lý tài liệu và danh mục tài liệu
- Hệ thống báo cáo vi phạm, cộng/trừ điểm
- Giao diện thân thiện và dễ sử dụng

## Công nghệ sử dụng
Dự án được xây dựng trên nền tảng .NET/C# (theo cấu trúc thư mục chứa các file .cs, Migrations).

## Cách chạy dự án
(Phần này có thể cập nhật chi tiết sau tùy theo cấu hình local của bạn).
